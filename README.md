# Articles Database

## Installation steps

### 1. Clone repository
```bash
git clone git@github.com:miyasinarafat/articles-database-frontend.git
```

### 2. Install the packages
```bash
npm install
```

### 5. Run application

```bash
npm run dev
```

## Access APIs
```bash
http://localhost:3000/
```
